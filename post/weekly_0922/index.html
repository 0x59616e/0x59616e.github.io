週記 0922

September 29, 2024

為了跟 nvim 培養感情，我想多寫點東西。

話雖如此，最近也沒什麼 code 可以寫，所以只好寫寫廢文。

m68k#

最近剛開學，而且又是碩一，閒得很，想回去弄 llvm m68k backend。

不過，這個後端現在只有一個人在維護，而且對方工作可能不輕鬆，很多 PR 懸在那裡都沒人 review。

所以我打算先從幫忙 review PR 開始。

跑步#

昨天 5K 終於跑進 26 分了，可喜可賀，要是哪天能跑進 20 分就更好了。

手機#

9/25 去辦離職，回程很厚臉皮地請主管載我回台大，結果就把手機忘在車上惹。

主管住新竹，下次上來台北是 10/3 號，我至少一個禮拜沒手機用。

到今天為止已經是第 4 天了，生活是有些不方便，但也還能過，但這種日子不太可能持續下去，生活中有太多部份都跟手機綁定在一起了。

像是登入 github 需要用手機驗證、買演唱會門票需要用手機認證。

我甚至差點衝動下訂一台新手機了，結果網路刷卡也需要用手機驗證。

這感覺還可以多談一些，不過先等我把手機拿回來吧。

模型#

最近正在做 TAMIYA 1/24 RZ34，這台的分件做得非常好，像是煞車卡鉗跟煞車碟盤，以往是合在一起的，現在是分開的，上色方便許多；座椅的分件也有注意到上色的方便性，整體來講是很不錯的模型。

田宮讓我喜歡的地方就在這裡，他們總是能找出可以改善的小地方，讓模型製作變得更加愉快。

話說他們年底要一次出 911 GT3 RS 和 1/48 F-35C，荷包要大失血了。

學業#

虛擬機器加簽上了，可喜可賀。

第一份作業是架環境和改一點 kvm，第一次動手改 linux，很新鮮。這門課是第一次讓我有「能來到這裡真是太好了」的念頭的課。

其它還有英文寫作基礎，雖然只是修來湊學分，但老師上課口條很好，而且共同教學館採光很好，風景明媚，上得很愉快。

相較之下，量子設計自動化特論就蠻折騰人了。這門課感覺比較偏向介紹性質吧，畢竟量子電腦這個領域很新，也不斷在進化中，也因此可能比較難規劃出有架構的課程，上了四個禮拜感覺還是學不太到什麼東西。

然後德田館通風是不是很差阿，上課很難不昏昏欲睡，真的很折磨人。
